# FapiaoPrint - 发票批量打印助手

🚀 一个专业的PDF电子发票批量打印网页工具，支持智能布局、批量处理和高质量输出。

## ✨ 功能特性
- 📤 拖拽上传PDF电子发票
- 🧩 智能布局（1页2张/1页4张）
- ⚙️ 可调节边距和裁剪线
- 💾 导出合并PDF文件
- 🖨️ 智能打印提示
- 🌐 纯前端，无需后端
- 📱 响应式设计

## 📖 使用说明
1. 拖拽PDF发票文件到页面
2. 选择布局模式（推荐1页4张横向）
3. 调整边距和裁剪线设置
4. 预览效果后导出或打印
5. 处理新文件前点击"清空重置"

## ⚠️ 注意事项
- 仅支持PDF格式电子发票
- 单次最多处理50个文件
- 建议使用Chrome/Edge/Firefox最新版
- 打印前建议先测试一页
